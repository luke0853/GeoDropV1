**Earn cryptocurrency through real-world geocaching adventures!**

GeoDrop is an innovative blockchain-based application that combines GPS-based geocaching with cryptocurrency rewards. Discover virtual "drops" at real locations, verify your presence with photos, and earn PixelDrop tokens through an engaging gamified experience.

## üîß Development Setup

### Prerequisites
- Node.js (for local development)
- MetaMask wallet
- BSC Testnet configuration

### Local Development
```bash
# Clone the repository
git clone https://github.com/luke0853/GeoDropV1.git

# Navigate to project directory
cd GeoDropV1

# Open in browser (or use a local server)
# For example, using Python:
python -m http.server 8000

# Or using Node.js:
npx serve .
```

### Configuration
1. Set up Firebase project
2. Configure AWS Rekognition
3. Set MetaMask to BSC Testnet
4. Update configuration files in `/config/`
